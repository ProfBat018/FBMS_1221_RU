## История
`JS` появился в 1995 году, его создал Брендан Эйх (Brendan Eich) в компании Netscape.
Интересный факт, они хотели назвать его `Mocha`, но потом переименовали в `LiveScript`, а потом в `JavaScript`.
Говорят что, все из-за популярности `Java` в то время. Вообще мы многим обязаны `Java`, но это другая история.

## Переменные
`JS` не строготипизированный язык, это значит что мы можем присваивать переменной разные типы данных.
Чтобы создать переменную существует два ключевых слова `var`, `let`, `const`. Про разницу чуть позже.
Очень важный момент заключается в том, что все переменные находятся внутри объекта `window`.

В `JS` есть спецификация создания перемнных. К нему относится например `hoisting`.
Это правило говорит о том, что все переменные нужно поднимать в начало области видимости.

Правила названия переменных:
1. Не должно начинаться с цифры
2. Не должно содержать спецсимволы кроме `$` и `_`
3. Не должно быть зарезервированных слов
4. Не должно быть пробелов
5. Не должно быть кириллицы 
6. Не должно быть kebab-case 
7. Не должно быть UPPER_CASE 
8. Не должно быть Title Case

Нужно использовать `camelCase` или `snake_case` для переменных и функций. `PascalCase` для классов.

### Есть три типа области видимости:
1. `global` - глобальная область видимости
2. `function` - область видимости функции
3. `block` - область видимости блока

#### Глобальная область видимости
Это значит что переменная будет доступна везде. Но есть один нюанс, если мы объявим переменную
внутри функции, то она будет иметь область видимости функции. Это значит что мы не сможем обратиться к ней
вне функции. 

#### Область видимости функции
Это значит что переменная будет доступна только внутри функции. 

#### Область видимости блока
Это значит что переменная будет доступна только внутри блока. Например внутри цикла `for` или `if`.

Разница между функцией и блоком в том, что функция создает область видимости, а блок нет.
Также функция создает новый контекст выполнения, а блок нет, пример будет в коде

### `var`
`var` - это ключевое слово для создания переменной. Оно имеет глобальную область видимости.
Также его можно переопределять и перезаписывать. Но есть один нюанс, если мы объявим переменную
внутри функции, то она будет иметь область видимости функции. Это значит что мы не сможем обратиться к ней
вне функции. Но если мы объявим переменную внутри блока, то она будет иметь глобальную область видимости.

### `let`
`let` - это ключевое слово для создания переменной. Оно имеет область видимости блока.

### `const`
`const` - это ключевое слово для создания переменной. Оно имеет область видимости блока.
Также его нельзя переопределять и перезаписывать. Но если мы объявим переменную
как объект, то мы можем менять его свойства, но не саму переменную.



## Типы данных
В `JS` есть 8 типов данных. Это `number`, `string`, `boolean`, `null`, `undefined`, `object`, `symbol`, `bigint`.
Также есть специальный тип `function`. Но это не тип данных, это объект. Но об этом позже. Также есть `NaN` и `Infinity`.

`undefined` - это тип данных, который означает что переменная не имеет значения. Пример: 
```js
let a;
console.log(a); // undefined
```

`null` - это тип данных, который означает что переменная имеет значение `null`. Пример: 
```js
let a = null;
console.log(a); // null
```

`NaN` - это тип данных, который означает что переменная имеет Not a Number. 
Такое значение получается когда мы пытаемся преобразовать строку в число, но это не получается. Пример: 
```js
let a = 'Hello';
console.log(Number(a)); // NaN
```

`Infinity` - это тип данных, который означает что переменная имеет бесконечное значение. Пример: 
```js
let a = 1 / 0;
console.log(a); // Infinity
```


## Условия
В `JS` есть три условных оператора. Это `if`, `else if`, `else`. Также есть тернарный оператор `?`.

## Операторы
В `JS` есть разница между операторами в других языках. Например `==` и `===`.
`==` - это оператор сравнения, он сравнивает только значения. 
`===` - это оператор строгого сравнения, он сравнивает значения и типы данных.

То есть если мы сравним `1 == '1'` то получим `true`, а если `1 === '1'` то получим `false`.

Также есть операторы `!=` и `!==`. Они работают также как и `==` и `===`, но наоборот.

Из логических операторов есть `&&`, `||`, `!`. Они работают также как и в других языках.
Из новых операторов есть `??`. Он возвращает первый операнд, если он не `null` или `undefined`, иначе второй.
Пример: 
```js
let a = null;
let b = 1;

console.log(a ?? b); // 1
```

## Циклы
В `JS` есть три цикла. Это `for`, `while`, `do while`. Также есть `for of` и `for in`.

Первые три вы знаете, а вот последние два нет. `for of` используется для перебора массивов.
Пример: 
```js

let arr = [1, 2, 3, 4, 5];

for (let item of arr) {
    console.log(item);
}
```

`for in` используется для перебора объектов. Пример: 
```js
let obj = {
    name: 'John',
    age: 20,
    city: 'Moscow'
};

for (let key in obj) {
    console.log(key, obj[key]);
}
```

К слову об объектах. JSON - это то что вы видите в примере выше. Это объект.

`for of` и `for in` даже взаимозаменяемы. Но есть один нюанс, `for of` не работает с объектами.


## Функции
В `JS` есть два типа функций. Это `function declaration` и `function expression`.

`function declaration` - это функция которая объявляется с помощью ключевого слова `function`.
Пример: 
```js
function sum(a, b) {
    return a + b;
}
```

`function expression` - это функция которая объявляется с помощью переменной.

Пример: 
```js
let sum = function(a, b) {
    return a + b;
}

sum(1, 2);

```

Также есть стрелочные функции. Они появились в `ES6`. Они имеют сокращенный синтаксис.
Пример: 
```js
let sum = (a, b) => a + b;
sum(1, 2);
```
